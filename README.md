# Neon Notes

A premium dark-themed notes app built with React, TypeScript, Tailwind CSS, Framer Motion, and local storage persistence.

## Features
- Create, edit, delete, and search notes
- Pin important notes
- Copy note content to the clipboard
- Responsive glassmorphism neon UI
- Automatic local storage persistence

## Scripts
- npm install
- npm run dev
- npm run build
